<h1>This is a guide on the AWS DevOps Agent API with AWS CLI</h1>

<p>This is a link to the AWS documentation on the AWS CLI API for the DevOps Agent: 
<a href="https://docs.aws.amazon.com/cli/latest/reference/devops-agent/">DevOps Agent API</a>
</p>

<h3>Commands to set up the agent</h3>

<p>To set up the agent you need to create a space, associate the relevant services to the 
space and enable the operator app for users to interact with the agent. You will also 
need to set up the relevant permissions for the agent space and the operator app.</p>

<h5>Create Space</h5>

<p>Command:</p>
<div>
    <pre>
        <code>
            <strong>bash</strong><br>
            aws devops-agent create-agent-space \
            --name <value> \
            --description <value> \
            --region <value>
        </code>
    </pre>
</div>

<p><a href="https://docs.aws.amazon.com/cli/latest/reference/devops-agent/create-agent-space.html">Link</a></p>

<h5>Associate AWS account - main account where the space is created</h5>

<p>Command:</p>
<div>
    <pre>
        <code>
            <strong>bash</strong><br>
            aws devops-agent associate-service \
            --agent-space-id <value> \
            --service-id aws \
            --region <value> \
            --configuration '{
                "aws": {
                    "assumableRoleArn": <value>,
                    "accountId": <value>,
                    "accountType": "monitor"
                }
            }'
        </code>
    </pre>
</div>

This is a guide on removing the AWS DevOps Agent with AWS CLI

This is a link to the AWS documentation on the AWS CLI API for the DevOps Agent: DevOps Agent API

Commands to remove the agent

To remove the agent we will follow the following steps:

  1. List all associations to the AgentSpace;
  2. Disassociate all associated services;
  3. Disable the OperatorApp;
  4. Delete the AgentSpace.

Step 1: list all associated services for the AgentSpace

Command:

        
            bash
            aws devops-agent list-associations \
            --agent-space-id <value>

Arguments:

  • agent-space-id - the id of the agent space.
AWS Docs

Step 2: remove associated services

Run the bellow command for each associated service returned by the command in step 1. Command: 

        
            bash
            aws devops-agent disassociate-service \
            --agent-space-id <value> \
            --association-id <value>

Arguments:

  • agent-space-id - the id of the agent space;
  • association-id - the id of the associated service.
AWS Docs

Step 3: disable the OperatorApp

Command:

        
            bash
            aws devops-agent disable-operator-app \
            --agent-space-id <value>

Arguments:

  • agent-space-id - the id of the agent space.
AWS Docs

Step 4: delete the AgentSpace

Command:

        
            bash
            aws devops-agent delete-agent-space \
            --agent-space-id <value>

Arguments:

  • agent-space-id - the id of the agent space.
AWS Docs
